# Graph Traversal Algorithms

## Breadth-First Search (BFS)
- **Time:** O(V + E)
- **Space:** O(V)
- **Use:** Shortest path in unweighted graphs, level-order traversal

## Depth-First Search (DFS)
- **Time:** O(V + E)
- **Space:** O(V)
- **Use:** Cycle detection, topological sort, pathfinding
