# Learning Resources for Data Structures & Algorithms

This curated list of resources will help you master DSA concepts and prepare for technical interviews.

## üìö Books

### For Beginners:
1. **"Grokking Algorithms"** by Aditya Bhargava
   - Visual and intuitive explanations
   - Great for complete beginners
   - Covers essential algorithms with illustrations

2. **"Cracking the Coding Interview"** by Gayle Laakmann McDowell
   - Interview-focused
   - 189 programming questions
   - Includes solutions and tips

3. **"Data Structures and Algorithms Made Easy"** by Narasimha Karumanchi
   - Problem-focused approach
   - Covers all major topics
   - Good for practice

### Intermediate:
4. **"Introduction to Algorithms" (CLRS)** by Cormen, Leiserson, Rivest, Stein
   - Comprehensive coverage
   - Rigorous and mathematical
   - Industry standard textbook

5. **"Algorithm Design Manual"** by Steven Skiena
   - Practical approach
   - War stories from real-world
   - Excellent for understanding applications

6. **"Algorithms"** by Robert Sedgewick & Kevin Wayne
   - Java implementations
   - Clear explanations
   - Companion to Coursera course

### Advanced:
7. **"The Art of Computer Programming"** by Donald Knuth
   - Comprehensive and deep
   - Mathematical rigor
   - Classic reference

8. **"Competitive Programming"** by Steven & Felix Halim
   - For competitive programmers
   - Advanced techniques
   - Problem-solving strategies

## üí° Tips for Using Resources

### For Interview Prep:
1. Focus on **LeetCode** medium problems
2. Follow a structured study plan (Blind 75 or Grind 75)
3. Do mock interviews on Pramp
4. Review company-tagged questions

### For Competitive Programming:
1. Start with **Codeforces** or **AtCoder**
2. Participate in contests regularly
3. Upsolve problems after contests
4. Learn advanced algorithms from CP-Algorithms

### For Learning:
1. Start with a good book (Grokking Algorithms)
2. Supplement with video lectures
3. Practice on HackerRank or LeetCode Easy
4. Visualize algorithms with VisuAlgo

## üìÖ Recommended Learning Path

### Month 1-2: Foundations
- Book: Grokking Algorithms
- Practice: HackerRank Easy
- Focus: Arrays, Strings, Basic DS

### Month 3-4: Intermediate
- Book: Cracking the Coding Interview
- Practice: LeetCode Easy-Medium
- Focus: Trees, Graphs, DP basics

### Month 5-6: Advanced
- Course: Coursera Algorithms
- Practice: LeetCode Medium-Hard
- Focus: Advanced DP, Graph algorithms

### Ongoing: Mastery
- Participate in contests
- Solve company-specific problems
- Mock interviews
- System design
